1 Data Splitting

Training Set (70%)

Used to train the model and learn patterns from the data

Validation Set (15%)

Used to tune hyperparameters and prevent overfitting

Test Set (15%)

Used for final evaluation of model performance

Why Split Data?

  • Training: Learn patterns and parameters from this subset
  • Validation: Tune hyperparameters without touching test data
  • Test: Unbiased evaluation of final model performance
  • Prevent Overfitting: Ensure model generalizes to unseen data
