How to Use the Demos

🎯 Classification

  1. Choose a dataset type and parameters
  2. Generate data points
  3. Select a classification algorithm
  4. Train the model and observe decision boundaries
  5. Analyze performance metrics

📈 Regression

  1. Select a function type to generate data
  2. Adjust noise level and number of points
  3. Choose regression algorithm
  4. Train and see the fitted curve
  5. Evaluate R², MSE, and MAE scores

🎪 Clustering

  1. Generate data with different distributions
  2. Select clustering algorithm
  3. Set number of clusters
  4. Run clustering and observe groupings
  5. Check silhouette score and inertia

🧠 Neural Networks

  1. Design network architecture
  2. Choose activation function and learning rate
  3. Build the network visualization
  4. Train and watch loss decrease
  5. Analyze parameter count and timing
